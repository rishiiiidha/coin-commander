# CoinCommander

CoinCommander is a command-line tool designed for cryptocurrency enthusiasts to perform various operations related to cryptocurrency information retrieval using the CoinMarketCap API. The application provides commands for managing API keys and checking cryptocurrency prices.

#  Installation


<h6>To install CoinCommander dependencies within the directory :</h6>

```bash
npm install 
```
<h6>To install CoinCommander dependencies , Node.js package globally on your system.</h6>

```bash
npm install -g
```
<h6> Starting to explore the features</h6>

```bash
coincommander
```

# Features
 <h3>API Key Management:</h3> 
 <ul>
   <li>Set a new API key for accessing the CoinMarketCap API.</li>
   <li>View the currently set API key.</li>
   <li>Remove the stored API key.</li>
 </ul>
 <h3>Crypto Check Commands:</h3>
<ul>
   <li>Check cryptocurrency prices using the CoinMarketCap API.</li>
   <li>Check specific coins and customize the cryptocurrency coins to check by providing a list of coin symbols as options.</li>
 </ul>

 <h1>Obtaining Your CoinMarketCap API Key</h1>
 <h6>Simple Steps for Accessing Cryptocurrency Data</h6>
 
 ### Create an account:

- Sign up on the [CoinMarketCap](https://coinmarketcap.com/).

### Find API section:

- Look for the API or Developers section after logging in.

### Generate API key:

- Click to generate a new API key.

### Provide info if needed:

- Fill in any required information.

### Copy and use securely:

- Copy the given API key and use it in your application, following CoinMarketCap's guidelines.
